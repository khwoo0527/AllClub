---
name: 메모리 저장 위치 판단 규칙
description: 범용 피드백은 .claude/memory에, 프로젝트 종속 정보는 프로젝트 메모리에 저장
type: feedback
---

중요한 피드백/학습이 생기면, 저장 전에 **어디에 저장할지** 판단하고 사용자에게 확인한다.

**Why:** 프로젝트에 종속되지 않는 범용 가이드라인을 프로젝트 메모리에 저장하면 다른 프로젝트에서 재사용 불가. 반대로 프로젝트 종속 정보를 .claude/memory에 넣으면 다른 프로젝트에서 혼란.

**How to apply:**
- 범용 (모든 프로젝트에 적용) → `.claude/memory/`에 저장 (git에 포함되어 다른 프로젝트로 복사됨)
- 프로젝트 종속 (이 프로젝트에만 해당) → 프로젝트 메모리에 저장
- 판단이 애매하면: "이 내용은 다른 프로젝트에서도 적용되는 범용 가이드인데, .claude/memory에 저장해서 범용으로 쓸 수 있게 할까요?"라고 사용자에게 확인
