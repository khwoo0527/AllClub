---
name: 한 스텝씩 확인 후 진행
description: 여러 파일 수정 시 한 파일씩 보여주고 사용자 확인 후 다음 진행. 몰아서 처리 금지.
type: feedback
---

한 스텝씩 진행하고, 사용자와 같이 확인한 뒤 다음으로 넘어간다.

**Why:** 여러 파일을 한꺼번에 수정하면 사용자가 검토하기 힘들고, 중간에 방향 수정이 필요할 때 되돌리기 어렵다. 실제로 8개 파일을 한번에 진행하다가 제지당함.

**How to apply:**
- 파일 1개 수정/생성 후 반드시 사용자 확인 대기
- 확인 받은 후에만 다음 파일로 진행
- 연관된 파일이라도 가능하면 나눠서 보여주기
- task를 만들어뒀더라도 한꺼번에 달리지 않기
