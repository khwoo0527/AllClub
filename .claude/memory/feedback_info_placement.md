---
name: 정보를 올바른 위치에 가이드
description: 사용자가 엉뚱한 곳에 정보를 넣으려 할 때 적극적으로 올바른 문서 위치를 안내해야 함
type: feedback
---

사용자가 상세 요구사항, 장기 목표 등을 CLAUDE.md나 대화에서 말할 때, 해당 정보가 들어갈 올바른 문서 위치를 적극적으로 안내해야 한다.

**Why:** 사용자가 CLAUDE.md에 PRD 수준의 상세 요구사항을 넣으려 했는데, 그대로 받아적으면 CLAUDE.md가 비대해지고 오히려 바이브 코딩 품질이 떨어진다.

**How to apply:**
- CLAUDE.md: 기술 스택, 빌드 명령, 프로젝트 구조, 환경 설정만
- docs/prd.md: 상세 요구사항, 비즈니스 로직, 사용자 시나리오, 장기 목표
- ROADMAP.md: Phase/Sprint 구조
- 사용자가 잘못된 곳에 정보를 넣으려 하면 "이건 PRD에 넣는 게 맞아요"라고 바로 안내
- 바이브 코딩 품질에 방해되는 정보는 빼야 한다고 솔직히 말하기
